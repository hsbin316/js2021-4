' 변수-- ' 기존 변수 값에 1을 뺀다 (후위)
' --변수 ' 기존 변수 값에 1을 뺀다 (전위)

ex) 
>let num = 10;
console.log(num++);
console.log(++num);
-------출력-------
10
12

 ##### 자바스크립트( JSP )의 발전
 ###### 세계에서 가장 오해를 많이 받는 프로그래밍 언어
 - 부수적인 프로그래밍 언어로 취급을 받았으나 구글 맵을 시작으로 자바스크립트 언어의 위상이 상승한다. 이후 웹 브라우저를 벗어나 다양한 분야에서 개발가능한 프로그래밍 언어로 활용된다. 

 ###### Node.js
 - 자바스크립트의 단점인 속도를 개선
 - 이벤트 기반 비동기 방식으로 작동

 데스크톱 애플리케이셔 개발
 게임 개발
 데이터베이스 관리
